    • splashscreen_logo.png (450px) → android/app/src/main/res/drawable-hdpi/
    • splashscreen_logo.png (600px) → android/app/src/main/res/drawable-xhdpi/
    • splashscreen_logo.png (900px) → android/app/src/main/res/drawable-xxhdpi/
    • splashscreen_logo.png (1200px) → android/app/src/main/res/drawable-xxxhdpi/
    • splash-icon.png (1200px) → assets/
  3. Rebuild the app with npx expo prebuild --clean then npm run android
